# CS50P - Problem Set 6: File I/O

This repository contains my solutions for **Problem Set 6** of Harvard University's **CS50's Introduction to Programming with Python (CS50P)**.

## 📚 Topics Covered

- File I/O in Python
- Reading and writing CSV files
- Command-line arguments
- Error handling
- Working with external libraries
- Image processing with Pillow

## 🛠️ Problems Included

### 📄 Lines
Counts the number of lines of code in a Python file, excluding blank lines and comments.

### 🍕 Pizza
Reads a CSV file containing pizza menu data and displays it in a nicely formatted table using the `tabulate` library.

### 📝 Scourgify
Converts names stored in `"last, first"` format into separate `first` and `last` columns in a new CSV file.

### 👕 Shirt
Overlays a shirt image onto an input image using the Pillow (`PIL`) library and saves the resulting image.

## 🎯 Learning Objectives

Through this problem set, I practiced:

- Handling files and CSV data in Python
- Processing command-line arguments
- Validating user input and handling exceptions
- Using third-party Python libraries
- Manipulating images with Pillow
